## 🚀 Quick Start Guide

### 1. Install Dependencies
```bash
cd frontend && npm install
cd .. && npm install
```

### 2. Configure Analytics
Update GA4 ID in `frontend/src/App.tsx`:
```typescript
useGoogleAnalytics({ trackingId: 'G-YOUR_ID' });
```

### 3. Start Development
```bash
npm run dev:all
```

### 4. Build for Production
```bash
npm run build
```
